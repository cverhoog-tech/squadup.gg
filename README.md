# SquadUp.gg v0.25

Functional UX rebuild.

Belangrijk:
- Desktop is nu een eigen command center, geen mindere kopie van mobiel.
- Game cards openen een echte game detail modal.
- Details tonen PvP/PvE/Co-op/Solo, beschrijving, owned/installed/want/max party.
- Instant Queue is vervangen door een Party Module met slots.
- Party slots kunnen gevuld/verwijderd worden.
- Je kunt squadleden toevoegen aan een party.
- Party invite plant een event en update de feed.
- Alle layouts blijven wisselbaar en gebruiken dezelfde werkende systemen.
