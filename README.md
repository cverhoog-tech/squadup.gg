# SquadUp.gg v0.30.1 Working Build

Deze build is een volledige werkende React/Vite app.

Inbegrepen:
- Premium login met 2x3 provider grid
- Offline Demo Mode
- localStorage demo persistence + logout
- Mobile dashboard
- Desktop dashboard
- Explore feed met PvP/PvE/co-op/solo info
- Game detail modal
- Party lobby
- Lobby chat
- Voice panel
- Squad/Friends
- Player profile drawer
- Avatar vault met 160 avatars
- Theme drawer
- Session memory / mini charts
- Runtime fallback om zwart scherm te voorkomen
